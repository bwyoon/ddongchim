## 똥침 게임
### html + css + javascript
### 입력한 프람프트
html+css+javascript 로 아래의 내용을 수행하는 게임을 만들어줘.
1. 일명 똥침 게임이야. 양손을 깍지 낀 다음 두 검지를 세워서 상대방의 항문을 찌르는 거야.
2. 상대방의 엉덩이는 위에서 연속적인 움직임으로 방향을 바꿔가며 움직여.
3. 찌르는 손은 맨 아래 위치하고 항문을 찌르는 검지는 위를 향하고 있어.
왼쪽 방향키와 오른쪽 방향키를 이용해 좌우로 좌우로 움직일 수 있어.
4. 스페이스바를 누르면 그 순간의 손 위치에서 손이 수직으로 위로 움직여서 0.2초 정도 후에 엉덩이가 위치하는 지점에 도착해.  이 때 손가락이 항문에 도착하면 득점을 하고 항문이 위치한 곳에서 벗어난 곳에 도착하면 실점을 해. 그리고 손은 다리 원래 위치로 0.2초 후에 되돌아와.
5. 게임은 시간제로 운영되고, 게임 난이도는 엉덩이의 움직임 빠르기로 결정해
